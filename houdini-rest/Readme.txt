
README
=======

1. Download File: houdini-rest-threejs.zip
2. Extract it to a Folder
3. Make sure that your System variable is correct so you can start hython in the commandline (cmd.exe)
	=> under Linux there should be a script: source houdini_setup
4. execute the bat File.
	=> under Linux open a Terminal and use the command: hython server.py
5. Load the Browser with the shown address: localhost:8000
